JS Variables:

Variables are containers for storing data values.

JavaScript uses reserved keyword var to declare a variable.

JavaScript uses let keyword to declare a variable.

JavaScript uses const keyword to declare a variable.

Variables are case sensitive.

Variables can be declared without a value. In such case, it will be initialized with a default value undefined.

Variables declared with let and const are hoisted to the top of the block, but not initialized.

Variables declared with var are hoisted to the top of the function, but not initialized.

Variables declared with let and const are block scoped.

Variables declared with var are function scoped.

Variables declared with const must be initialized with a value.

Variables declared with const cannot be reassigned.

Variables declared with const cannot be redeclared.

JS Escape Sequence Characters:

JavaScript uses the \ (backslash) as an escape characters

  • \'    single quote
  • \"   double quote
  • \    backslash
  • \n   new line
  • \r   carriage return
  • \t   tab
  • \b   backspace
  • \f   form feed
  • \v   vertical tab (IE < 9 treats '\v' as 'v' instead of a vertical tab ('\x0B'). If cross-browser compatibility is a concern, use \x0B instead of \v.)

String Methods:

Strings are immutable. This means that once a string is created, it is not possible to modify it. However, it is still possible to create another string based on an operation on the original string.

1.String Length

Helps to find the length of a string.

1. let name = "Shihab";    
2. console.log(name.length);
    
//Output: 6

2.String toUpperCase()

Converts the string to uppercase letters.

1. let name = "Shihab";
2. console.log(name.toUpperCase());

//Output: SHIHAB

3.String toLowerCase()

Converts the string to lowercase letters.

1. let name = "Shihab";
2. console.log(name.toLowerCase());

//Output: shihab

4.String replace()

Replaces a specified value with another value in a string.

1. let name = "Shihab";
2. console.log(name.replace("Shihab", "Faysal"));

//Output: Faysal

5.String concat()

Joins two or more strings, and returns a new joined strings.

1. let name1 = "Shihab";
2. let name2 = "Faysal";

3. console.log(name1.concat(" is a friend of " + name2));

//Output: Shihab is a friend of Faysal

6.String trim()

Removes whitespace from both ends of a string.

1. let name = "     Shihab     ";
2. console.log(name1.trim());

//Output: Shihab

7.String slice()

Extracts a part of a string and returns a new string. This method takes 2 parameters: the starting index (position), and the ending index (position).Also one parameter would also do. If a parameter is negative, the position is counted from the end of the string.

1. let name = "Shihab";
2. console.log(name1.slice(1,3));

//Output: hi

Template Literals:

This is a concept where we can insert variables in console by using "backticks- ` ` ".

For instance:

1. let boy1 = "Shihab";
2. let boy2 = "Faysal";

3. console.log(`${boy1} is a friend of ${boy2}`);

//Output: Shihab is a friend of Faysal
Using backticks will similarly print the output as a string like we use double quotes to print the strings. but the only difference is that with backticks we can insert the variables as shown in the example above. ** We can insert variables in the template literals.This is called string interpolation.
