JS Variables:

Variables are containers for storing data values.

JavaScript uses reserved keyword var to declare a variable.

JavaScript uses let keyword to declare a variable.

JavaScript uses const keyword to declare a variable.

Variables are case sensitive.

Variables can be declared without a value. In such case, it will be initialized with a default value undefined.

Variables declared with let and const are hoisted to the top of the block, but not initialized.

Variables declared with var are hoisted to the top of the function, but not initialized.

Variables declared with let and const are block scoped.

Variables declared with var are function scoped.

Variables declared with const must be initialized with a value.

Variables declared with const cannot be reassigned.

Variables declared with const cannot be redeclared.

String Methods:

Strings are immutable. This means that once a string is created, it is not possible to modify it. However, it is still possible to create another string based on an operation on the original string.

1.String Length

Helps to find the length of a string.

1. let name = "Shihab";    

2. console.log(name.length);
    
//Output: 6

2.String toUpperCase()

Converts the string to uppercase letters.

1. let name = "Shihab";

2. console.log(name.toUpperCase());

//Output: SHIHAB

3.String toLowerCase()

Converts the string to lowercase letters.

1. let name = "Shihab";

2. console.log(name.toLowerCase());

//Output: shihab

4.String replace()

Replaces a specified value with another value in a string.

1. let name = "Shihab";

2. console.log(name.replace("Shihab", "Faysal"));

//Output: Faysal

5.String concat()

Joins two or more strings, and returns a new joined strings.

1. let name1 = "Shihab";
2. let name2 = "Faysal";

3. console.log(name1.concat(" is a friend of " + name2));

//Output: Shihab is a friend of Faysal

6.String trim()

Removes whitespace from both ends of a string.

1. let name = "     Shihab     ";

2. console.log(name1.trim());

//Output: Shihab

7.String slice()

Extracts a part of a string and returns a new string. This method takes 2 parameters: the starting index (position), and the ending index (position).Also one parameter would also do. If a parameter is negative, the position is counted from the end of the string.

1. let name = "Shihab";

2. console.log(name1.slice(1,3));

//Output: hi

Array:

Arrays are used to store multiple values in a single variable.

Arrays are a special type of objects.

Arrays use numbers to access its "elements". //Numbers meant the indeces of the array.

Arrays are mutable. //That means array elements can be changed.

Arrays are passed by reference.

Arrays are compared by reference.

Arrays are converted to strings by joining all the elements with commas (,) in between.

It is a common practice to declare arrays with the const keyword.

Array declaration:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

Or,

const numbers = [50, 60, 40, 80];

We can also create an array, and then provide the elements:

1. const fruits = [];

2. fruits[0] = "Banana";
3. fruits[1] = "Orange";
4. fruits[2] = "Apple";
5 .fruits[3] = "Mango";

6. console.log(fruits);

//Output: ["Banana", "Orange", "Apple", "Mango"]

Accessing array elements

We can access the elements of an array with the help of it's index.

Array indexes start with 0.

Example:
1. const fruits = ["Banana", "Orange", "Apple", "Mango"];

2. console.log(fruits[0]);

//Output: Banana

**If we give an index which doesn't exist in the array then the output will be Undefined.

Array Methods:

There are many methods for arrays. Some of them are:

1.Array length

Helps to find the length of an array.

1. const fruits = ["Banana", "Orange", "Apple", "Mango"];

2 .console.log(fruits.length); //Output: 4

Converts an array to a string of (comma separated) array values.

4.Array pop()

Removes the last element of an array, and returns that element.

**The pop() method returns the value that was "popped out":

Example:
1. let fruits = ["Banana", "Orange", "Apple", "Mango"];

2 .console.log(fruits.pop())

//Output: Mango

5.Array push()

Adds new elements to the end of an array, and returns the new length.

**The push() method returns the new array length:

Example:
1. let fruits = ["Banana", "Orange", "Apple", "Mango"];

2. console.log(fruits.push("Kiwi"))

//Output: 5

So pop() function returned the new length of the array.

If we console 'fruits' now we will see the pushed elemend at the end of the array:

2. console.log(fruits)
                
//Output: ["Banana", "Orange", "Apple", "Mango", "Kiwi"]

To remember: pop() function excludes the last element of the array and returns the poped element but push() adds the element at the end of the array.but just returns the length of the new array.

6.Array shift()

The shift() method removes the first array element and "shifts" all other elements to a lower index.

**The shift() method returns the value that was "shifted out":

Example:
1. let numbers = [50, 60, 40, 70];

2. console.log(numbers.shift());

//Output: 50

7.Array unshift()

The unshift() method adds new items to the beginning of an array, and returns the new length.

**The unshift() method returns the new array length.

Example:
1. let numbers = [50, 60, 40, 70];

2. console.log(numbers.unshift(10));

//Output: 5

So shift() function returned the new length of the array.

If we console 'numbers' now we will see the unshifted elemend at the starting of the array:

2. console.log(numbers);

//Output: [10, 50, 60, 40, 70]

To remember: shift() function excludes the first element of the array and returns the shifted element but unshift() adds the element at the beginning of the array.but just returns the length of the new array.

8.Array delete()

The delete operator is designed to remove properties from JavaScript objects, which arrays are objects.

**The delete operator does not affect the length property.

**The delete operator leaves undefined in the place where the deleted array element used to be.

**The delete operator is designed to be used on object properties. It has no effect on variables or functions.

Example:

Let's say we want to delete the second element of the array 'numbers'.

So we will do:

1. let numbers = [50, 60, 40, 70];

2. delete numbers[1];

3. console.log(numbers);

//Output: [50, undefined, 40, 70]

So we can see that the second element of the array is deleted and the place is filled with undefined.

But the length of the array will still be 4.

9.Array concat()

Joins two or more arrays, and returns a copy of the joined arrays.

**The concat() method does not change the existing arrays. It always returns a new array.

Example:

Let's say we want to join two arrays 'numbers' and 'numbers2'.

So we will do:

1. let numbers = [50, 60, 40, 70];
2. let numbers2 = [10, 20, 30];

3. let numbers3 = numbers.concat(numbers2);

4. console.log(numbers3);

//Output: [50, 60, 40, 70, 10, 20, 30]

*We can concatenate more than two arrays also.

Example:

Let's say we want to join three arrays 'numbers', 'numbers2' and 'numbers3'.

So we will do:

1. let numbers = [50, 60, 40, 70];
2. let numbers2 = [10, 20, 30];
3. let numbers3 = [1, 2, 3];

4. let numbers4 = numbers.concat(numbers2, numbers3);

5. console.log(numbers4);

//Output: [50, 60, 40, 70, 10, 20, 30, 1, 2, 3]

10.Array sort()

The sort() method sorts an array alphabetically.

**The sort() method sorts the items of an array in place. The sort order can be either alphabetic or numeric, and either ascending (up) or descending (down).

**By default, the sort() method sorts the values as strings in alphabetical and ascending order.

**This works well for strings ("Apple" comes before "Banana"). However, if numbers are sorted as strings, "25" is bigger than "100", because "2" is bigger than "1".

**Because of this, the sort() method will produce an incorrect result when sorting numbers.

**You can fix this by providing a "compare function" (See "Parameter Values" below).

**The purpose of the compare function is to define an alternative sort order.

**The compare function should return a negative, zero, or positive value, depending on the arguments, like:

This sort() function changes or modifies the source array into a new array as it changes the positions of the elements

Example:

Let's say we want to sort the array 'numbers'.

So we will do:

Without compare function:

1. let numbers = [50, 60, 40, 70];

2. numbers.sort();

3. console.log(numbers);

//Output: [40, 50, 60, 70]

With compare function:

Ascending order:

Let's say we want to sort the array 'numbers' in ascending order.

So we will do:

1. let numbers = [50, 60, 40, 70];

2. let compare = (a, b) => {
       return a - b;
   }

3. numbers.sort(compare);

4. console.log(numbers);

//Output: [40, 50, 60, 70]

Descending order:

Let's say we want to sort the array 'numbers' in descending order.

So we will do:

1. let numbers = [50, 60, 40, 70];
                        
2. let compare = (a, b) => {
        return b - a;
   }
                        
3. numbers.sort(compare);
                        
4. console.log(numbers);
                        
//Output: [70, 60, 50, 40]

Template Literals:

This is a concept where we can insert variables in console by using "backticks- ` ` ".

For instance:

1. let boy1 = "Shihab";
2. let boy2 = "Faysal";

3. console.log(`${boy1} is a friend of ${boy2}`);

//Output: Shihab is a friend of Faysal
Using backticks will similarly print the output as a string like we use double quotes to print the strings. but the only difference is that with backticks we can insert the variables as shown in the example above. ** We can insert variables in the template literals.This is called string interpolation.
